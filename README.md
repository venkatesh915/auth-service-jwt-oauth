# Auth Service Backend (JWT + Google & LinkedIn SSO)

A secure and scalable backend authentication system built with JWT-based authentication and OAuth2 social login support for Google and LinkedIn.

This project demonstrates modern authentication practices used in production-grade applications, including token-based auth, third-party OAuth integration, and secure session handling.

---

## 🚀 Features

- 🔐 JWT-based authentication (access & refresh tokens)
- 🔑 Google OAuth2 Login (SSO)
- 💼 LinkedIn OAuth2 Login (SSO)
- 👤 User registration & login system
- 🛡️ Password hashing with bcrypt
- 🔄 Token refresh mechanism
- 🚪 Secure logout (token invalidation)
- 🌐 RESTful API architecture
- ⚙️ Environment-based configuration

---

## 🏗️ Tech Stack

- **Backend:** Node.js / Express (or your framework)
- **Auth:** JWT, OAuth2
- **Database:** MongoDB / PostgreSQL (update accordingly)
- **Security:** bcrypt, helmet, cors
- **OAuth Providers:** Google, LinkedIn
